<h1>Gestion des plaintes internes</h1>
 <h2>Description</h2>
Le module Gestion des Plaintes Internes est un module personnalisé développé pour l’ERP Odoo 17, destiné à faciliter la gestion, le suivi et le traitement des plaintes internes des employés au sein d’une organisation.
Ce module s’intègre directement avec le module Human Resources (HR) d’Odoo, assurant ainsi une gestion cohérente des données liées aux employés.
<h2>Fonctionnalités principales</h2>
 °Création et gestion des plaintes internes des employés.
 °Association de chaque plainte à un employé existant.
 °Classification des plaintes par type (harcèlement, discipline, autre).
 °Suivi de l’état des plaintes (nouveau, en cours, traité).
 °Vues liste et formulaire.
 °Gestion des droits d’accès via le système de sécurité d’Odoo
<h2>Objectifs du module</h2>
 °Centraliser la gestion des plaintes internes.
 °Améliorer le suivi et la traçabilité des plaintes.
 °Faciliter le travail des responsables et du service des ressources humaines.
 °Offrir une base extensible pour l’ajout de fonctionnalités.
<h2>Environnement technique</h2>
 °Odoo : version 17
 °Base de données : PostgreSQL
 °Déploiement : Docker
 °Langages : Python, XML
